# Справочник API

## Класс Calculator

Основной класс для математических операций.

### `add(a, b)`

Складывает два числа.

**Параметры:**

- `a` (float) - первое число
- `b` (float) - второе число

**Возвращает:**

- `float` - сумма a и b

**Пример:**

```python
from src.calculator import Calculator

calc = Calculator()
result = calc.add(5, 3)  # Возвращает 8.0
```
